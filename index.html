Skyscraper Builder!

* You will be given an order to build a skyscraper with a certain number of floors at the start of the game.

* There are four cranes below. By clicking on a crane, you will add a specific number of floors to your skyscraper.

* You win the game by building your skyscraper with the exact number of floors ordered. You lose the game if the number of floors exceeds that number.

* The number of floors each crane builds is unbeknownst to you until you use it.

* Each time you begin to build a new skyscraper, the number of floors each crane builds will change.
